# pythree.c
Search for pythagorean triples (*a² + b² = c²*)

Usage example:
```bash
$ ./pythree 100 1
# Output:
# Mode: filtered triples
# (3, 4, 5)
# (5, 12, 13)
# (7, 24, 25)
# (9, 40, 41)
# (11, 60, 61)
# (13, 84, 85)
```

Build:
```bash
$ make
```
